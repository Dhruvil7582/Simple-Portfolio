## ✨ Features

- **Hero Section** — Introduction with name, role, and call-to-action buttons
- **Navigation Bar** — Smooth links to Home, About, Portfolio, Blog, and Contact
- **Social Links** — Quick access to Instagram, LinkedIn, and GitHub
- **Contact Section** — Two-panel contact form with name, email, and message fields
- **Responsive Design** — Adapts to all screen sizes
- **Hover Animations** — Smooth transitions on buttons, nav links, and social icons

## 🚀 Getting Started

### View Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/Dhruvil7582/portfolio.git
   ```

2. **Navigate into the folder**
   ```bash
   cd portfolio
   ```

3. **Open in your browser**
   ```bash
   open index.html
   # or just double-click index.html
   ```

No build tools or dependencies required — it's pure HTML & CSS!
